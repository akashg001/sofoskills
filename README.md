# sofoskills
# Features
## Listed below are all the feature included in the project

###  Game Feature
#### 5 stages of game
* presence of mind
* 3D perspective view
* concentration
* logic building
* 3D movement ability

#### 7 soft skills assessed
* Critical Thinking
* Memory
* Patience
* Focus
* Stress Management
* Problem Solving
* Eye for Detail

#### Authentication
 * User Sign up
 * User Sign in
 * Admin login
 
 ####  Admin can add the questions
 #### timer to keep track of the time taken by the user to answer the question
